AI-Powered Conversation Chatbot

An automated chatbot system built to handle customer inquiries and schedule appointments through natural conversations utilizing LLM's.

Project Overview
Project Goals

This project aims to streamline customer interactions using AI and telephony services. It allows users to engage with the chatbot via phone calls, respond to queries, and even schedule appointments by leveraging ChatGPT and a web-based audio framework for efficient conversation handling.

Key Features
  • Natural Language Processing for real-time conversations
  • Twilio API integration for call handling
  • Google Calendar integration for scheduling appointments
  • Custom audio prompts and Text-to-Speech (TTS) responses
  • Flexible architecture allowing local or cloud-based deployment
Technical Implementation
Phone Connection & API Integration

The AI chatbot connects to real users through telephone, analyzing voice commands and just as easily responding. The chatbot uses multiple APIs, such as Azure for voice recognition, ElevenLabs for voice generation, and Google Calendar API to retrieve available times and confirm appointments with customers.

RAG Process

A RAG process using TF-IDF performs efficient querying on scraped web-pages -- in this case, 150+ scraped company web pages. This allows the chatbot to provide accurate and relevant information based on the company's knowledge base.

Conversation Flow

Twilio integrations enable a structured flow for the customer's call, from greeting to handling specific inquiries and finalizing appointments or call terminations. The system can handle invalid responses and repeat prompts when necessary.

Text-to-Speech & Recognition

Using ElevenLabs TTS and Azure audio recognition, the system can convert text to lifelike speech to answer questions and accurately process customer responses. This creates a natural and engaging conversation experience.
