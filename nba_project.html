NBA Analytics Project

Leveraging advanced data engineering techniques to analyze NBA player statistics and uncover valuable insights.

Project Overview
Project Goals

The primary goal of this project is to analyze NBA player performance and determine the most valuable players for the Detroit Pistons. By leveraging both local and cloud-based data engineering solutions, the project empowers users with flexible options for processing and analyzing data at scale. The inclusion of neural networks for predictive analysis and a PowerBI dashboard for visual representation ensures a comprehensive solution for basketball analytics.

Key Achievements
  • Developed an 8-step ETL Medallion pipeline using Azure SQL Server, Azure Data Lake, Azure Data Factory, Databricks, and Synapse database
  • Analyzed 12,000+ player observations using Time-Series (LSTM) and MLPNN predictions
  • Achieved a predictive neural net for NBA wins with a 66% accuracy (industry standard at 60-70%)
  • Maintained an organized codebase of 6 large modules on GitHub for a custom PyPi package
Technical Implementation
Cloud-Based Data Engineering (Azure)

Using Microsoft Azure, this project offers scalable data ingestion, storage, and processing capabilities. The cloud setup allows for real-time data analysis with seamless integration across tools and services, perfect for larger datasets and teams working remotely.

Neural Networks for Predictive Analysis

The project employs advanced neural network architectures for predicting player performance. Using historical data, these models are trained to forecast future player efficiency and contributions to team success. The models focus on key metrics such as points per game, assists, rebounds, and win shares.

Local Data Engineering

The local setup provides a controlled environment for smaller-scale analysis. It allows you to run all data processes and machine learning tasks on your local machine, providing greater flexibility for development and iteration before scaling up to the cloud if needed.

PowerBI Dashboard for Visualization

Data insights are visualized through a PowerBI dashboard, enabling users to easily explore trends and performance metrics. The dashboard presents key data, such as player efficiency ratings, win predictions, and comparative analyses between different players.

View dynamic insights on player performance, team comparisons, and predictive outcomes. The dashboard is fully interactive, allowing users to filter data based on time frames, players, or performance categories.
