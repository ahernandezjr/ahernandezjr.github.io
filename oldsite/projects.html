A collection of my recent projects showcasing data engineering, AI, and machine learning work.

NBA Analytics Project

The NBA Analytics project analyzes player performance to uncover insights for the Detroit Pistons. This project uses advanced data engineering techniques both on cloud (Azure) and local setups. It employs neural networks to predict player stats and outcomes, offering a comprehensive solution for sports analytics. Additionally, it features a PowerBI dashboard for intuitive data visualization.

Developed an 8-step ETL Medallion pipeline using Azure SQL Server, Azure Data Lake, Azure Data Factory, Databricks, and Synapse database, ending with Power BI dashboard visualizations. Analyzed 12,000+ player observations using Time-Series (LSTM) and MLPNN predictions. Achieved a predictive neural net for NBA wins with a 66% accuracy (industry standard at 60-70%). Maintained an organized codebase of 6 large modules on GitHub for a custom PyPi package.

AI-Powered Conversation Chatbot

The AI-Powered Conversation Chatbot is designed to handle customer interactions for a company, providing information and scheduling appointments. This project integrates the Twilio API for phone call capabilities, ChatGPT for natural language understanding, Azure and ElevenLabs for voice handling, and TF-IDF for relevant content extraction. The chatbot connects over 150+ webpages and intelligently responds to customer queries using prompt engineering.

Developed a chatbot using Flask, Twilio, ChatGPT, ElevenLabs, and Google APIs to interact with potential customers, provide company-related information, and schedule appointments. Designed a RAG process using TF-IDF for efficient querying on 150+ scraped company web pages. Integrated a Large Language Model (LLM) through prompt engineering for accurate responses. Maintained a Docker image to deploy as an Azure App Service utilizing CI/CD.

Fungi Article Acquisition Project

The Fungi Article Acquisition project automates the collection of research papers related to fungal species. By scraping research article websites, it builds a rich dataset for bioinformatics analysis, making the study of fungi more efficient and scalable. It uses Python scripts, APIs, and database management techniques to store, organize, and process the information.
